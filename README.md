#Фирменный редактор кода с поддержкой программирования блоками и кодом для контроллера ZERO и платформ Arduino 

#Программируй при помощи блоков
В режиме "Блоки" программирование происходит при помощи широкой библиотеки блоков. Для многих блоков доступна подсказка для подключения. 

#Программируй по классике!
В режиме "Код" доступен полноценный редактор кода, с поддержкой подсветки синтаксиса, автодополнениями,  подсказками из установленных библиотек, подсветкой синтаксических ошибок и предупреждениями.

#Загружай программу прямо из редактора
Прямо в редакторе КонтрБагКОД доступна компиляция и загрузка прошивки, а так же полноценный монитор порта. 

#Прогоаммируй на любой платформе
Благодаря полноценному менеджеру плат поддерживаются все ядра доступные для Arduino IDE!

#Запуск на Linux
КонтрБагКОД пока не собирается в AppImage. Но предоставлеяется в виде gz архива.
Для запуска на Linux выполните следующие действия в папке приложения:
- Желательно поместить папку с приложением в домашнюю папку пользователя
- Пометьте исполняемым arduino-cli (находится в папке resources)
```
cd resources
chmod +x arduino-cli
```
- Пометье исполняемым и запустите без песочницы kontrbugcode
```
chmod +x kontrbugcode 
./kontrbugcode --no-sandbox
```
